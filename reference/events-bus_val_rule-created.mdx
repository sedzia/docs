# EVENT: "EVENTS.BUS\_VAL\_RULE.CREATED"HTTP method: POST

The `EVENTS.BUS_VAL_RULE.CREATED` event indicates that a new business validation rule has been successfully created in Voucherify. This event provides, among others, the following details about the newly created rule:

* ID,
* Conditions,
* Logic.

This event is used in a webhook configured in Project settings in Voucherify dashboard.

<Info>
  This page documents only the event. If you need more details about the webhook payload data that includes this event, go to [Webhook v2024-01-01](/reference/introduction-to-webhooks "Introduction to webhooks v2024-01-01") page.
</Info>
